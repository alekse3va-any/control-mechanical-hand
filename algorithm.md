# Алгоритм работы

1. Пользователь открывает веб-интерфейс.
2. Пользователь нажимает кнопку «Сжать руку» или «Разжать руку».
3. JavaScript отправляет запрос на Flask-сервер.
4. Flask отправляет команду Arduino через USB Serial.
5. Arduino управляет сервоприводом.
6. Arduino считывает кнопку в ладони.
7. Arduino возвращает JSON-ответ.
8. Сайт показывает состояние руки и наличие объекта.
